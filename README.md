# Site de Voyage

## Description du projet
Ce projet consiste à développer un site web de voyage en **HTML**, **JavaScript** et **Bootstrap 5.3.9**.  
Il permet de simuler un site marchand de voyages avec un **panier** pour gérer les commandes et un système de **filtres** pour trier les voyages selon vos préférences.

---

## Pour commencer
Le site utilise un jeu de données pour afficher les voyages disponibles et permet aux utilisateurs de :
- Ajouter et retirer des voyages dans un panier.
- Trier et filtrer les voyages selon différents critères (prix, destination, durée, etc.).
- Visualiser les voyages populaires et les offres spéciales.

---

## Fonctionnalités principales
- **Affichage dynamique** des voyages à partir d’un jeu de données.
- **Système de panier** pour gérer les voyages sélectionnés.
- **Filtres** pour rechercher les voyages selon différents critères.
- **Interface responsive** grâce à Bootstrap 5, adaptée aux ordinateurs, tablettes et mobiles.
- Mise en avant des **voyages populaires et offres spéciales**.

---

## Pré-requis
- Navigateur moderne : Chrome, Firefox, Edge, Safari…
- **Bootstrap 5.3.9**
